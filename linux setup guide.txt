CLONE THE REPO:
git clone https://github.com/Seigneur-Machiavel/contrast

NO NVM? INSTALL IT:
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.1/install.sh | bash
THEN RESTART OR ENTER:
export NVM_DIR="$([ -z "${XDG_CONFIG_HOME-}" ] && printf %s "${HOME}/.nvm" || printf %s "${XDG_CONFIG_HOME}/nvm")"
[ -s "$NVM_DIR/nvm.sh" ] && \. "$NVM_DIR/nvm.sh"

GET LAST NODE VERSION:
nvm install node

NAVIGATE TO PROJECT FOLDER AND INSTALL DEPENDENCIES:
cd contrast && npm i

RUN:
node node/run.mjs
Options:
-na stresstest (spend a lot of Txs)
-pk [private_key] (setup a private without dashboard access)

(Optional) Use "nohup" if you want the process in the background (usefull for ssh connection)
nohup node node/run.mjs -na dashboard -pk [private_key] > output.log 2>&1 &
